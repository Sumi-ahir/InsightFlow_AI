## Features

- JWT Authentication (Login / Register)
- Google OAuth Login
-  Real-time AI Chat System
-  Multiple Chat Sessions
-  Smart Chat Title Generation
-  Chat History Persistence
-  Optimized Backend APIs
-  Responsive UI
-  -Use Tavily for latest Search
